## KolpaqueClientElectron

Electron version of Kolpaque Client

Multiplatform GUI for livestreamer/streamlink. Supports KLPQ stream service, twitch and youtube (with your own api key).
Works on Windows, Mac, Linux.

Run from source.
```
git clone
npm install
npm start
```
