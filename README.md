## KolpaqueClientElectron

Electron version of Kolpaque Client **React Edition**

Multiplatform GUI for Streamlink. Supports KLPQ stream service, twitch and youtube (with your own api key).
Works on Windows, Mac, Linux.

Run from source.
```
git clone
yarn install
yarn run start
```

Run React.
```
yarn run start-dev
```

Build.
```
yarn run build
```
